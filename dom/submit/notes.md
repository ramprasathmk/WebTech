
## Events

Events are intractions or actions happen during user and browser.

1. Keyboard Events 

2. Mouse Events

3. Form Events

4. Window Events

These are the events done by the user.

## Keyboard Events

These are the events done by the user using `keyboard events`.

- `keydown`: while press down key this event will be triggered.

- `keyup`: while press up key this event will be triggered.

## Mouse Events

These are the interactions done by user using keybiard events

- `click`: when we click the event will be executed.

- `dblclick`: when we click the event will be executed.

- `mouseover`: when we place the cursor over the element this event will be executed.

## Form Events

These are the interactions happens inside the form.

- `submit`: while submitting the form it will be triggered.

- `reset`: wihle reset the form it will clear all the fields and triggered.

- `input`: when we do changes in the input it will be triggered immediately.

- `change`: when we do change in the input but once changes completed it will be triggered. or when it blur it will be triggered.

## Window Events

While changes in window

- `load`: when the window loads it will be triggered.

- `beforeunload`: when the window or document before to load immediately it will be triggered.

- `resize`: when the window gets resized it will be triggered.

- `scroll`: when the window gets scrolled it will be triggered.
