# Combinator Selector

  - It is a combination of 2 selectors.
  - Based on the relation b/w 2 elements the CSS will target the elements.
  - They are: 
    1. Descendent Selector ()
    2. Direct Child Selector (>)

### **Descendent Selector: -->**
  - It will target all the childern, grandchildren, great grandchildren and so on.
  - The symbol used is space ( ).

### **Direct Child Selector: -->**
  - It will target only the children but not grandchildren, great grandchildren and so on.
  - The symbol used is greater than (>).

### **Adjacent Sibling Selector: -->**
  - Targets the `very next immediate sibling`.
  - Works only if the selected element comes `right after` the other element.
  - The symbol is +.

### **General Sibling Selector: -->**
  - Targets all the siblings appear `after the Selected element`.
  - The Symbol is ~.
